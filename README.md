# Country Quiz App

#### Site demo: https://tender-fermat-8d1d02.netlify.app/

#### A trivia game that ask random questions about country capitals and flags.

## How to get start

#### Download the zip or clone the repo

#### `cd country-quiz-app`

#### Run `npm install`

#### Run `npm start`
